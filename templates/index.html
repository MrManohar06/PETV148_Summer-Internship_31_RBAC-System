Role-Based Access Control (RBAC) System

A Flask reference implementation of NIST SP 800-162 RBAC with JWT authentication, fine-grained per-resource permissions, and privilege-escalation detection, aligned with the OWASP Access Control Cheat Sheet.

Roles

Admin, Manager, User, Guest — each mapped to a ranked hierarchy per NIST core RBAC.

Fine-Grained Permissions

Every route checks a specific (resource, operation) pair against the database — not just a role name.

Escalation Detection

JWT role claims are cross-checked against the database on every request; mismatches are logged as CRITICAL events.
